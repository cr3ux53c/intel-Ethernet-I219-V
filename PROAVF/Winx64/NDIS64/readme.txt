The driver package files in this folder can be used to install the Adaptive Virtual Function (AVF) driver on the following guest partition operating systems:
   *  Microsoft Windows Server* 2012 R2
   
The driver package supports devices based on the following controllers:
  * Intel(R) Ethernet Controller X710
  * Intel(R) Ethernet Controller XL710
  * Intel(R) Ethernet Network Connection X722
  * Intel(R) Ethernet Controller XXV710
